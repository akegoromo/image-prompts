# 並行世界の404年

### #1. 404年、自由の女神が一歩踏み出す世界線
- 場面: ニューヨークの港に立つ自由の女神像が、台座から片足を大きく前に踏み出し、前方の海面をのぞき込んでいる。
- 一言: 「女神が一歩踏み出した先は、まだデータが敷かれていない何もない海（404）でした。」

### #2. 404年、パンゲア超大陸が再合体した世界線
- 場面: すべての大陸が磁石のようにギュッと一つに引き寄せられ、地球の半分に巨大な「404」の形をした超大陸が形成されている。
- 一言: 「世界のすべての陸地が合体した結果、このURLの居場所が押し潰されて消滅しました。」

### #3. 404年、スフィンクスが「お代わり」を習得した世界線
- 場面: 砂漠に鎮座するスフィンクスが、前足を可愛らしくチョンと上げて「お代わり」のポーズをしている。目の前には巨大な犬の餌皿。
- 一言: 「謎解きを出すのをやめて芸を覚えましたが、探しているページはお代わりできません。」

### #4. 404年、重力が90度横に傾いた世界線
- 場面: 街中の人々や車、ビルから生える植物がすべて真横（右方向）に向かって引っ張られ、横倒しになったまま生活している。
- 一言: 「世界の軸が90度回転しました。お探しのページは画面の右外へ落ちていきました。」

### #5. 404年、すべての文字が「フォント：毛筆」になった世界線
- 場面: 近未来の都市の看板やスマホの画面、道路の標識に至るまで、すべての文字が超豪快な「男気の毛筆体」で書かれている。
- 一言: 「四〇四。其の頁（ページ）、此の電脳空間（サーバー）には存在せず。」

### #6. 404年、ジャンケンに「ビーム」が追加された世界線
- 場面: 子供たちがジャンケンをしているが、グー・チョキ・パーの他に、手から放射線状のビームを出す4つ目の手つき（404の手）を出して勝負が混乱している。
- 一言: 「あいこが続きすぎた結果、ジャンケンの法則（URL）自体がバグを起こしました。」

### #7. 404年、人類の主食が「消しゴム」になった世界線
- 場面: 高級レストランで、ウエイターが銀の皿に乗った最高級の「MONO消しゴム」を恭しくサーブしている。客はナイフとフォークを持っている。
- 一言: 「この世界の主食はすべてを消し去る性質があり、このページも綺麗に消されました。」

### #8. 404年、すべての山がプリンでできている世界線
- 場面: 登山家たちがスコップを持って富士山のような巨大なプリンの山を登り、カラメルソースの頂上を目指して食べている。
- 一言: 「美味しい世界ですが、お探しのURLは誰かがスプーンですくい取ってしまいました。」

### #9. 404年、恐竜が絶滅せずサラリーマンになった世界線
- 場面: ティラノサウルスが窮屈そうなスーツを着て、小さなノートパソコンを短い前足で必死にタイピングしている。
- 一言: 「恐竜のIT進化が早すぎたため、この古いページは化石（404）になりました。」

### #10. 404年、猫が世界の言語を統一した世界線
- 場面: 国連の会議場で、各国の代表たちが真面目な顔をして「ニャー」「ミャオ」「フシャァー」とだけ発言している。
- 一言: 「世界が可愛い鳴き声で満たされたため、エラーテキストも『ニャー』しかありません。」

- 場面: 雨の日。街ゆく人が皆、お椀のように逆さまに開いた傘を差しており、傘の中に雨水がタプタプに溜まっている。
- 一言: 「受け止める準備は万端ですが、降ってくるはずのデータ（ページ）がありません。」

- 場面: 最高額紙幣の真ん中に、ごく普通のTシャツを着て頼りなさそうに微笑んでいる「佐藤さん（仮名）」の顔が印刷されている。

- 場面: ファッションショーのランウェイを、全身プチプチでできたドレスを着たモデルが歩いている。観客が我慢できずにプチプチ潰している。

- 場面: スズメやカラスが真面目な顔でトボトボと交差点を歩き、その上空をランドセルを背負った小学生が平泳ぎのスタイルで優雅に飛んでいる。
